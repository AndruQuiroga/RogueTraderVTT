# Content Policy

This repo must not include copyrighted book text, art, or other proprietary content.

Allowed

- Original text written for this project.
- Data models, code, and configuration needed to run the system.
- References to public URLs and citations that do not reproduce copyrighted content.

Not Allowed

- Rulebook text, tables, or images copied from published Rogue Trader or related books.
- Scanned PDFs or extracted assets.
- Packs or compendiums that embed copyrighted descriptions or art.
