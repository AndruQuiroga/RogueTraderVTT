# Roll20 Reference Assets

Files copied from the Roll20 Rogue Trader Improved sheet for mapping and layout reference.

- `reference/roll20/RogueTraderImproved.html`: Roll20 sheet HTML, includes roll buttons and sheet worker scripts.
- `reference/roll20/RogueTraderImproved.css`: Roll20 sheet CSS styles.
- `reference/roll20/sheet.json`: Roll20 sheet metadata (authors, preview, and file names).

Notes

- These files are reference-only for migration planning and attribute/roll formula mapping.
- The editable Rogue Trader PDF is not included here; keep any licensed PDFs local and do not commit them.
