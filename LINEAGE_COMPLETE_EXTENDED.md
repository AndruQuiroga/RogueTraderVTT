## Implementation Notes

### Perilous Choice Special Cases

The Perilous Choice talent requires special runtime implementation:

```javascript
// Check for unsanctioned psyker flag
if (actor.items.some(i => i.flags?.rt?.isUnsanctionedPsyker)) {
  // Use Renegade Psyker table for Psychic Phenomena
  const psychicStrengthRow = "renegade"; // Instead of "sanctioned"
}

// Prevent Astropaths/Navigators from taking it
if (actor.system.career === "astropath" || actor.system.career === "navigator") {
  ui.notifications.warn("Astropaths and Navigators cannot take Perilous Choice");
  return false;
}
```
